
# URL Shortener Node.js

This repo contains a basic URL shortener API in Node.js. 

## Technologies

- Node.js
- Express
- Postgresql




# Endpoints
#### Create short link

```http
  POST /
```

| Body | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `longUrl`      | `string` | **Required**  |

#### Use short link to be redirected

```http
  GET /{code}
```
